# quibbler

A library to make acceptance testing feel amazing in clojure.

## Getting Started

### Build

Build the application with:

```
  lein build
```

### Testing

Test the application with:

```
  lein test
```

### Documentation

Build the documentation with:

```
  lein codox
```
